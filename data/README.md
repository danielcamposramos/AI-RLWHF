# Data Directory

Central home for all dataset artifacts powering AI-RLWHF experiments. Mix of curated corpora, user resources, synthetic assets, and teacher-student honesty traces lives here.

## Key Structures
- `raw/`: Immutable upstream corpora plus manifests describing provenance and licensing.
- `processed/`: Normalized datasets, including `honesty_logs/` JSONL or Parquet slices generated by evaluator plugins.
- `synthetic/`: Multi Vibe Coding In Chain generations with explicit prompt and reward metadata.
